# PROJECT_CONTEXT.md

# Aquard

## Project Overview

Aquard is an open-source premium Home Assistant custom card for monitoring water quality.

The project aims to transform technical water measurements into a beautiful, intuitive interface that can be understood within seconds.

Aquard should never feel like a collection of sensors.

It should feel like a premium appliance.

---

# Vision

Water monitoring should be simple.

Most users do not care about raw sensor values.

They want answers to simple questions:

- Is my water safe?
- Can I use it?
- Do I need to do something?

Aquard should answer those questions first.

Technical information should always be available, but should never overwhelm the user.

---

# Target Audience

Aquard is designed for both beginners and advanced users.

Beginner users should immediately understand the status of their water without knowing anything about pH, ORP or EC.

Advanced users should be able to expand the interface and inspect every available measurement.

---

# Supported Water Profiles

Aquard is designed around reusable water profiles.

Current planned profiles:

- Spa
- Pool
- Pond
- Aquarium
- Custom

Each profile defines:

- ideal measurement ranges
- health calculations
- available controls
- recommendations
- profile-specific terminology

The UI should remain profile independent.

---

# User Experience Goals

Aquard should provide:

- immediate understanding
- minimal cognitive load
- beautiful visual design
- premium feeling
- responsive interaction
- consistent behavior

The first screen should answer:

"Is my water ready?"

---

# Home Assistant Integration

Aquard is a native Home Assistant custom card.

It should integrate naturally with:

- Sections dashboards
- Masonry dashboards
- Kiosk Mode
- Themes
- HACS

Aquard should feel like a natural extension of Home Assistant.

---

# Design Goals

Aquard should look like a premium control panel rather than a dashboard.

The design language is inspired by:

- premium consumer electronics
- modern vehicle interfaces
- luxury spa control panels
- clean industrial design

The interface should feel calm, elegant and trustworthy.

---

# Technical Goals

The project values:

- maintainability
- modularity
- responsiveness
- performance
- accessibility
- clean architecture

over feature count.

---

# Long-Term Vision

Aquard should become the preferred Home Assistant interface for monitoring water quality regardless of hardware manufacturer.

It should support any Home Assistant entities through configuration.

Aquard should never depend on a specific sensor brand or water controller.

---

# Mission Statement

Aquard transforms complex water data into clear, actionable information that anyone can understand within seconds.
